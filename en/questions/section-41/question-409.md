#### ❓ Question 409: Copying, renaming, or deleting files

- Create a file named `copyme.txt` and write a line to it.
- Copy the file to `copied.txt`.
- Rename `copied.txt` to `renamed.txt`.
- Finally, delete `renamed.txt`.
