#### âœ… Answer 21: Using `if`, `else if`, and `else`

**Explanation:**

Conditional branching in Rust uses `if`, `else if`, and `else` to control program flow based on conditions.

```rust
fn main() {
    let num = 5;
    if num > 0 {
        println!("Positive");
    } else if num < 0 {
        println!("Negative");
    } else {
        println!("Zero");
    }
}
```
