#### âœ… Answer 25: Conditional counting using `while`

**Explanation:**

A `while` loop continues as long as its condition is true.

```rust
fn main() {
    let mut n = 1;
    while n <= 5 {
        println!("n: {}", n);
        n += 1;
    }
}
```
